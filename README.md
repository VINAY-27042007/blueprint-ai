# 🚀 BlueprintAI

BlueprintAI is an AI-powered project blueprint generator built using React and Google Gemini AI.

Users can enter a project idea and instantly generate a complete project blueprint including:

* Abstract
* Problem Statement
* Objectives
* Modules
* Methodology
* Expected Outcomes
* Future Scope

---

## ✨ Features

### 🤖 AI-Powered Blueprint Generation

Generate complete project documentation using Google Gemini AI.

### 📄 PDF Export

Download generated blueprints as PDF files.

### 📝 TXT Export

Download generated blueprints as text files.

### 📋 Copy Blueprint

Copy the generated blueprint with one click.

### ⏳ Loading State

Displays a loading message while AI is generating content.

### 🎯 Dynamic Project Generation

Supports various project ideas such as:

* Library Management System
* Hospital Management System
* College Management System
* Food Delivery System
* Attendance Management System
* And many more...

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/blueprint-ai.git
```

Move into the project:

```bash
cd blueprint-ai/frontend
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the frontend folder:

```env
VITE_GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```
